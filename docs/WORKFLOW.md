# Работа с проектом (кратко)

## Что есть в корне папки

| Файл | Роль |
|------|------|
| `ОТКРЫТЬ_ЛЕНДИНГ.bat` | Основной цикл для проверки: освободить порт 3000 → `npm run build` → открыть браузер → `npm run start`. |
| `ОБНОВИТЬ_ДАННЫЕ.bat` | Запуск `python scripts\export_audit_v2.py` из родительской папки **`Аналитик данных`** (пересборка JSON). После этого снова `ОТКРЫТЬ_ЛЕНДИНГ.bat`. |
| `СОБРАТЬ_ФИНАЛ.bat` | То же без «жёсткого» режима окна заголовка: сборка и `start`, порт **3000** перед стартом тоже чистится. |

## Связка с родительской директорией

- CSV/Excel → скрипт: `..\scripts\export_audit_v2.py` → артефакт: `src\data\audit.json`.

## Особые файлы верстки

- `ChartsSectionGate` загружает Recharts только в браузере (`dynamic` + `ssr: false`), чтобы при `next build` не было предупреждений о нулевом размере контейнеров диаграмм.
